# Ghostty

---

## Configurações

| Opção | Valor | Efeito |
|---|---|---|
| `theme` | `omtheme` | Tema de cores ativo |
| `font-family` | JetBrains Mono | Fonte principal |
| `font-size` | 28 | Tamanho da fonte |
| `font-feature` | `-calt`, `-dlig`, `-liga` | Ligaduras desabilitadas |
| `cursor-style` | block | Cursor em bloco |
| `cursor-style-blink` | true | Cursor piscante |
| `background-opacity` | 1.0 | Sem transparência |
| `background-blur-radius` | 0 | Sem blur |
| `mouse-hide-while-typing` | true | Esconde o mouse ao digitar |
| `gtk-single-instance` | true | Uma única instância GTK |
| `gtk-tabs-location` | bottom | Abas na parte inferior |
| `gtk-wide-tabs` | false | Abas compactas |
| `window-padding-x` | 16, 8 | Padding horizontal (esq: 16, dir: 8) |
| `window-padding-y` | 0 | Sem padding vertical |
| `window-padding-balance` | true | Padding balanceado |

---

## Temas disponíveis

| Tema | Status | Característica |
|---|---|---|
| `omtheme` | ativo | Fundo `#0f0f14`, tons roxo/rosa, cursor `#d6d1ff` |
| `tokyonightlocal` | disponível | Alto contraste, fundo `#121212`, cores vívidas |

Para trocar de tema, edite a linha `theme = ...` em `ghostty/config`.
