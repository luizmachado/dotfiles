# i3wm

Ambiente de janelas tiling com tema Tokyo Night.

Configuração gerenciada via `include` — o `~/.config/i3/config` gerado pelo
`i3-config-wizard` não é substituído. As customizações são injetadas por
arquivos incluídos.

---

## Arquivos

| Arquivo dotfiles | Symlink | Conteúdo |
|-----------------|---------|----------|
| `i3/theme.conf` | `~/.config/i3/theme.conf` | Fonte, cores, bordas, gaps, picom |
| `i3/luiz.conf` | `~/.config/i3/luiz_dotfiles.conf` | Keybindings customizados |
| `i3/i3status.conf` | `~/.config/i3status/config` | Status bar com ícones Nerd Font |
| `i3/rofi/config.rasi` | `~/.config/rofi/config.rasi` | Launcher com tema Tokyo Night |

---

## Keybindings customizados

| Atalho | Ação |
|--------|------|
| `$mod+d` | Rofi — launcher de aplicativos |
| `$mod+p` | Move workspace para monitor da esquerda |
| `$mod+Shift+p` | Captura de tela (Flameshot GUI) |
| `$mod+Shift+d` | Desativa protetor de tela e DPMS (modo apresentação) |
| `$mod+Shift+a` | Reativa protetor de tela e DPMS (padrão 5 min) |

---

## Tema (Tokyo Night)

**Fonte:** `JetBrainsMono Nerd Font 10` (títulos) / `9` (barra)

**Cores de janela:**

| Estado | Borda | Fundo |
|--------|-------|-------|
| Focada | `#7aa2f7` (azul) | `#3d59a1` |
| Inativa | `#24283b` | `#24283b` |
| Desfocada | `#1a1b26` | `#1a1b26` |
| Urgente | `#f7768e` (vermelho) | `#f7768e` |

**Bordas:** `pixel 2` (sem title bar)

**Gaps:** `inner 8` / `outer 4` (requer i3 ≥ 4.22)

## Status bar (i3status)

Lido automaticamente de `~/.config/i3status/config`.

| Item | Ícone | Descrição |
|------|-------|-----------|
| CPU | `󰻠` | Uso percentual |
| Memória | `󰍛` | Usado / Total |
| Disco | `󰋊` | Espaço disponível em `/` |
| Rede (ethernet) | `󰈀` | IP local |
| Rede (wifi) | `󰖩` | Nome da rede (ESSID) |
| Bateria | `󰁹` / `󰂄` | Percentual + tempo restante |
| Horário | `󰥔` | `dd/mm/yyyy HH:MM` (formato BR) |

## Rofi

Launcher moderno em substituição ao dmenu, acessado por `$mod+d`.

- **Modo:** `drun` (aplicativos instalados)
- **Fonte:** JetBrainsMono Nerd Font 12
- **Tema:** Tokyo Night (fundo `#1a1b26`, destaque `#3d59a1`, borda `#7aa2f7`)
- **Config:** `~/.config/rofi/config.rasi` → `i3/rofi/config.rasi`

---

## Compositor (picom)

Iniciado automaticamente via `exec --no-startup-id picom --daemon` no `theme.conf`.
Habilita transparência e é a base para efeitos visuais futuros.

---

## Nerd Font

A JetBrainsMono Nerd Font é instalada em `~/.local/share/fonts/NerdFonts/`
pelo `install.sh` (download do GitHub Releases). É necessária para renderizar
os ícones na barra de status e no rofi.

Para verificar se está instalada:
```bash
fc-list | grep "JetBrainsMono Nerd"
```
